All about Display Grid

Setting Up files Video

In this video, he first teaches a shortcut of how to create multiple child elements at the same time. You do this by typing .(div class name).item{#} * (a number). Then press Tab to make your main div plus it's elements. Then, he teaches how to create a variable in CSS by typing :root {} and inside the curly brackets, type --(variable name): (value); . In order to use the variable, type var(--(variable name)).

Grid Basics

In this video we learn about grid-template-column and grid-template-row. The grid-template-column controls how many items are in each row. The columns go horizontal. grid-template-row controls how many rows there are, which are vertical. You can also control the height and weight using both of these. This is done in CSS.

Responsive Design

In this video we learn how to make sure the boxes stay in order in different screen sizes using Responsive Design.

AutoFlow

In this video, we learn how to use Autoflow. Autoflow is used to automatically create another column/row when you add a new item.

Span

In this video we learned how to use span to alter the size and color of an individual section. If you want a section to take up the place of 2 columns, you would type in CSS: grid-column: span 2; . If you want a section to take up 3 rows, you would type in css: grid-row: span 3; .
